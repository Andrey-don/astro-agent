# CLAUDE.md — astro-agent

## Что это

Агент для генерации статей на сайт astro-obzor.ru.
Пишет три типа статей: новостные, научпоп, смешанные.
Результат сохраняется в папку `articles/` — оттуда копируешь в WordPress.

## Ключевые файлы

| Файл | Назначение |
|------|-----------|
| `brief.md` | Сайт, аудитория, рубрики, форматы |
| `tone-of-voice.md` | Стиль статей — прикреплять к каждому запросу |
| `content-plan.md` | Недельный план и банк тем |
| `articles/` | Готовые статьи — копировать в WordPress |

## Агенты

| Агент | Модель | Роль |
|-------|--------|------|
| Ресёрчер | Sonnet 4.6 | Собирает факты по теме |
| Писатель | Sonnet 4.6 | Пишет статью |
| Редактор | Sonnet 4.6 | Правит и улучшает |
| SEO | Haiku 4.5 | Title, description, slug, ключевые слова |

## Цикл генерации

Тема → Ресёрчер → Писатель → Редактор → SEO → файл в articles/

## Запуск

```
python -m bot.main
```

## SEO-агент (claude-seo)

Установлен глобально: `C:\Users\profi\.claude\skills\seo\`

| Команда | Что делает |
|---------|-----------|
| `/seo audit https://astro-obzor.ru` | Полный аудит сайта |

| `/seo schema <url>` | Schema-разметка |
| `/seo technical <url>` | Технический SEO |
| `/seo content <url>` | Качество контента (E-E-A-T) |
